# README
Rails app for searching in public github repos
Uses ruby 3.2.2 and the rails project is created with --minimal and --skip-activerecord flags

*** setup
clone the repo and run
#+BEGIN_SRC shell
    bundle
    rails s
#+END_SRC

*** run tests
#+BEGIN_SRC shell
    bundle exec rspec
#+END_SRC

*** run linters
#+BEGIN_SRC shell
    bundle exec rubocop
#+END_SRC
